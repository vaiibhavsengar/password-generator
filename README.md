# 🔐 Apram - Password Generator

**Apram** is a simple and secure password generator built using **HTML, CSS, and JavaScript**. It helps users create strong and random passwords to enhance their online security and protect their accounts.

## ✨ Features

* 🔒 Generate strong and random passwords
* ⚙️ Customize password length
* 🔤 Support for uppercase and lowercase letters
* 🔢 Include numbers and special characters
* 📋 Copy passwords to clipboard instantly
* ⚡ Fast and lightweight
* 📱 Responsive and user-friendly design

## 🔄 How It Works

1. 🚀 Launch the application.
2. ⚙️ Select your preferred password settings.
3. 🔐 Generate a secure password instantly.
4. 📋 Copy the password and use it for your accounts.
